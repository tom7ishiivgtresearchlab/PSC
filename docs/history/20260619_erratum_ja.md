# 正誤訂正ノート：バリオン非対称性の式について

**著者：** Tsutomu Ishii（石井 努）、Independent Researcher, Nagano, Japan
**発行日：** 2026年6月19日
**対象：** Planck Sphere Cosmology (PSC) 初版（2026年6月16日, Zenodo DOI: 10.5281/zenodo.20736648）

## 1. 訂正の概要

PSC初版において、バリオン非対称性パラメータ η の最終式に数式表記の誤りがあった。本ノートでこれを訂正する。導出の過程で用いた数値（α³/4π² が観測値の約16倍、ゆえに16で割る）は正しく、誤っていたのは「16で割る」操作を数式に書き下す際の分母の係数のみである。

## 2. 誤りの内容

初版の記載：η = α³/(16π²)

この式をそのまま計算すると、2.46×10⁻⁹ となり、観測値 6×10⁻¹⁰ の約4倍（誤差約300%）であって、「6.1×10⁻¹⁰」とは一致しない。

## 3. 正しい式

導出文「第一候補 α³/4π² ≈ 9.8×10⁻⁹ は観測値の約16倍大きいため、16で割る」を数式に正しく反映すると、分母は 4π² をさらに16倍した 64π² となる：

$$\eta = \frac{\alpha^3}{64\pi^2} \approx 6.15\times10^{-10}$$

**検算：**

| 式 | 値 | 観測値（6.1×10⁻¹⁰）との誤差 |
|----|-----|----------------------------|
| α³/(4π²)（第一候補） | 9.84×10⁻⁹ | ― |
| α³/(16π²)（初版の誤り） | 2.46×10⁻⁹ | 約303% |
| α³/(64π²)（訂正後） | 6.15×10⁻¹⁰ | 0.85% |

すなわち、初版が主張した「観測値との一致」という結論自体は正しく、それを表す数式が誤っていた。正しい式 α³/(64π²) は観測値と誤差0.85%で一致する。

## 4. 係数 64 の扱いについて（重要な但し書き）

初版では分母の係数を「16 = 2⁴ = ディラック構造（4×4行列の16成分）」と解釈していた。しかし正しい分母の係数は 64 であり、16ではない。したがって初版の「ディラック構造との対応」という解釈は、そのままでは成立しない。

64 は複数の形に分解できる：

- 64 = 4 × 16 （第一候補の係数4 と、観測値に整合させるための因子16=2⁴ の積）
- 64 = 2⁶
- 64 = 4³

これらはいずれも数学的に正しい表現だが、どの分解が物理的に要請されるかは現時点で未解決である。重要な点として、係数64は観測値からの逆算によって得られた数であり、第一原理から演繹されたものではない。

したがって本訂正版では、η = α³/(64π²) を「観測値とよく一致する経験的な関係式（empirical relation）」として位置づけ、係数64の物理的必然性の解明を今後の課題とする。これは初版が「16の理論的必然性」を残された課題として挙げていたことと整合する姿勢である。
